# radio-pwa
Pakistan FM &amp; News Radio – PWA. Mobile-friendly app streaming FM 100, FM 91, FM 90.4, FM 99, BBC World Service, Radio Pakistan, VOA Urdu &amp; Deewa. Single player auto-stops previous station, installable on Android, responsive, and works offline via service worker.

### Hosting Instructions

1. Push this folder to a **GitHub repository**  
2. Enable **GitHub Pages** in repository settings  
3. Open the HTTPS URL in Chrome  
4. Tap **Add to Home Screen** → App ready for Android install  

### Notes

- Some FM stations require public stream URLs. If a station doesn’t play, verify the URL.  
- Streaming audio uses **network data**, approx 20–30MB per hour at typical news FM quality (48–64kbps).
